好多好多的幸福， 都是有你陪在身边。宝贝，我爱你 所以我希望给你最好的， 不允许你受任何的委屈，不忍心看你不开心，新的窗口，即使闹别闹了我也先想到的是让你好好的，情绪可以慢慢来调整，但是我不会让你受委屈，先低头的决定是我，我不宠你谁宠你呢。
